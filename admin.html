One-Time Setup: GitHub Token

To publish posts directly from this page, we need a GitHub Personal Access Token. This is a one-time step — it's saved in your browser and you'll never need to do it again.

  1. Go to github.com → Settings → Developer Settings → Personal Access Tokens → Tokens (classic)
  2. Click Generate new token (classic)
  3. Give it a name like "SCV Blog Admin"
  4. Set expiration to No expiration (or 1 year)
  5. Check the repo scope (full repository access)
  6. Click Generate token and copy it
Your token is saved only in this browser. It is never sent anywhere except directly to GitHub's API.
