Admin Login

Access is granted via your GitHub Personal Access Token. Your token is only stored locally in your browser session — it's never sent to any server other than GitHub.

How to get your token

  1. Go to GitHub → Settings → Personal access tokens
  2. Click Generate new token (classic)
  3. Give it a name like SCV Admin and check the repo scope
  4. Copy the token and paste it below

Token must have repo scope and access to Steel-City-Visuals/SCV-Website.

You'll stay signed in until you manually sign out.
